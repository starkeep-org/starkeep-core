## About meta-docs

"Meta-docs" is an experimental approach for todo management and codebase documentation. 
I've been using it mainly to help manage todos for starkeep. It was more convenient
to commit this to the repo, and I thought the documents might be of interest to some.

If you're interested in the meta-docs system, see: https://github.com/amkoller/meta-docs
